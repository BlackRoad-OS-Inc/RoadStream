# RoadStream — Sovereign Media Server

> Forked from [Jellyfin](https://github.com/jellyfin/jellyfin). Self-hosted streaming on your Pi fleet.

**RoadStream** serves video, music, and photos from your own hardware. No subscriptions, no tracking, no content restrictions.

## Features
- **Video streaming** with HLS/DASH adaptive bitrate
- **Music library** with playlists and radio
- **Photo galleries** with AI-generated tags
- **Transcoding** via Hailo-8 accelerator (52 TOPS)
- **Multi-user** with per-user libraries

## Deploy
```bash
docker compose up -d  # runs on Octavia
# Access at stream.blackroad.io
```
